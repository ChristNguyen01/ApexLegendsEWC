# Apex Legends EWC Analysis

This project analyzes professional Apex Legends EWC Finals data to evaluate player performance.

## Goals
- Measure player consistency
- Identify clutch performers
- Compare performance metrics

## Features
- Custom consistency score
- Clutch performance analysis
- Data visualization

## How to Run
1. Download dataset from Kaggle
2. Place CSV files in `data/raw/`
3. Run notebook

## Key Insights
- Consistency is a major factor in performance
- Clutch situations separate top players
- High damage does not always guarantee high placement
